**Machine Learning-Based Prediction of Wine Quality Using Physicochemical Properties**

**Overview**

This repository contains the QM640 Data Analytics Capstone project titled Machine Learning-Based Prediction of Wine Quality Using Physicochemical Properties.

The project aims to develop and evaluate multiple machine learning regression models for predicting wine quality using physicochemical characteristics from the UCI Wine Quality Dataset. The study compares the predictive performance of different regression algorithms to identify the most accurate and interpretable model for supporting objective wine quality assessment and data-driven decision-making.

**Objectives**
- Predict wine quality using machine learning regression models.
- Compare the predictive performance of multiple regression algorithms.
- Identify the physicochemical properties that most significantly influence wine quality.
- Improve model interpretability using SHAP and, where appropriate, LIME explainability techniques.
- Demonstrate the business value of an explainable machine learning framework for wine quality assessment and production optimization.

**Dataset**

Source:
https://archive.ics.uci.edu/dataset/186/wine+quality

The study uses the Wine Quality Dataset from the UCI Machine Learning Repository, containing laboratory-measured physicochemical properties and quality scores for Portuguese Vinho Verde red wines.
