# QM640 Data Analytics Capstone

## Project Title

Machine Learning-Based Prediction of Wine Quality Using Physicochemical Properties

## Project Description

This repository contains the project artifacts developed for the QM640 Data Analytics Capstone. The study aims to develop and compare machine learning regression models to predict wine quality using physicochemical properties from the UCI Machine Learning Repository.
